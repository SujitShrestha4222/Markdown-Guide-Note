# 💬 Blockquotes in Markdown:
Blockquotes are used to highlight important text, notes, or messages.



---



## ✅ Basic Blockquote:
```md id="g7k2dn"
> This is a blockquote
```

> This is a blockquote



---



## ✅ Multi-line Blockquote:
```md id="q8m1sx"
> This is line one  
> This is line two
```

> This is line one
> This is line two



---



## ✅ Nested Blockquote:
```md id="r4t9yb"
> Main quote  
>> Nested quote
```

> Main quote
>
> > Nested quote

## ✅ Styled Tips (Very Useful 🚀):
