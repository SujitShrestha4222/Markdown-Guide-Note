# 📊 Tables in Markdown:
Tables help you organize data clearly.



---



## ✅ Basic Table:
```md
| Name  | Age | Role       |
|-------|-----|------------|
| Sujit | 22  | Developer  |
| Ram   | 25  | Designer   |
```

| Name  | Age | Role      |
| ----- | --- | --------- |
| Sujit | 22  | Developer |
| Ram   | 25  | Designer  |



---



## ✅ Alignment:
```md
| Left Align | Center Align | Right Align |
|:-----------|:------------:|------------:|
| Text       | Text         | Text        |
```

| Left Align | Center Align | Right Align |
| :--------- | :----------: | ----------: |
| Text       |     Text     |        Text |



---



## 💡 Tips:
* Use `:` to control alignment
* Keep tables simple (don’t overload)
* Great for comparisons



---



## 🎯 Real Use Case:
```md
| Model          | Accuracy | Speed |

```
