# ✍️ Text Formatting in Markdown:
Markdown lets you style your text easily.



---



## ✅ Bold & Italic:
```md
**This is bold**
*This is italic*
***Bold + Italic***
```

**This is bold**
*This is italic*
***Bold + Italic***



---



## ✅ Strikethrough:
```md
~~This is removed text~~
```

~~This is removed text~~

## ✅ Inline Code:
```md
Use `print()` in Python
```

Use `print()` in Python

## ✅ Code Block:
```md 
python

```

## 💡 Tips:
* Use backticks (`) for code, not quotes (' ')
* Use code blocks for better readability
* Great for sharing code on GitHub
