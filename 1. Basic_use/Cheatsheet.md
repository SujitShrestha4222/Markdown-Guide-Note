# ⚡ Markdown Cheatsheet:
Quick reference for Markdown syntax.



---



## 🔤 Headings:
```md
# H1
## H2
### H3
```



---



## ✍️ Text Formatting:
```md
**Bold**
*Italic*
~~Strikethrough~~
`inline code`
```



---



## 📋 Lists:
```md
- Item
- Item
  - Subitem

1. First
2. Second
```


---



## 🔗 Links & Images:
```md
[Google](https://google.com)
![Image](image-url)
```



---



## 💻 Code Blocks:
```md
\`\`\`python
print("Hello")
\`\`\`
```



---



## 📊 Tables:
```md
| Name | Age |
|------|-----|
| A    | 20  |
```



---



## 💬 Blockquote:
```md
> This is a quote
```



---



## ✅ Task List:
```md
- [x] Done
- [ ] Not done
```



---



## 🏷️ Badge:
```md
![Badge](https://img.shields.io/badge/Example-blue)
```
