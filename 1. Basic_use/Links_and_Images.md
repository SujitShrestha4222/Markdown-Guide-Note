## ✅ Relative Links / Images

You can link files in your repo:

```md id="m2b4tp"
[README](README.md)
![Local Image](images/my-image.png)
```

[README ho ma](/README.md)<---click this
![Local Sujit's Image](/1.%20Basic_use/profile%20picture.jpg)
### I am Sujit🧑🤳.


---

## 💡 Tips

* Always add **Alt text** for accessibility. It refers to the Alt attribute in Markdown images: e.g: [Alt text](your_url)
* Titles appear when you hover on GitHub
* Relative paths make your repo portable

---

## 🎯 Real Use Case

* Link to your project website
* Show screenshots of your project
* Add logos or badges
