# ✅ Task Lists in Markdown:
Task lists are checkboxes used to track progress.



---



## ✅ Basic Syntax:
```md id="v9q3rt"
- [ ] Task not done
- [x] Task completed
```
we can click on boxes to tick/untick.
where x = tick
&
use "-" or "*" at beginning of []


* [ ] Task not done
* [x] Task completed



---



## ✅ Example TODO List:

- [x] Learn Markdown basics  
- [x] Practice formatting  
- [ ] Build a project README  
- [ ] Add screenshots

- [x] Learn Markdown basics

## 💡 Tips:
* Works best on GitHub
* You can click to check/uncheck
* Useful for tracking progress
